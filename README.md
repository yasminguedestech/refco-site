# RefCo — Mentoria Criativa & Técnica

Site oficial da **RefCo**, empresa de mentoria criativa e técnica fundada por [Yasmin Guedes](https://www.linkedin.com/in/yasmin-guedes-0101/) e [Ryan Basque](https://www.linkedin.com/in/ryan-basque/).

Ajudamos pessoas a construírem sua presença digital do zero — site, portfólio ou layout próprio — com qualidade profissional e identidade autêntica.

---

## Sobre o projeto

Site estático de uma página (`index.html`) desenvolvido com HTML, CSS e JavaScript puro — sem frameworks, sem dependências externas.

### Seções
- **Hero** — apresentação e chamada principal
- **Como funciona** — passo a passo do processo de mentoria
- **Serviços** — o que a RefCo oferece
- **Projetos** — trabalhos realizados
- **Planos** — Grátis, Ref., Create. e Go.
- **FAQ** — dúvidas frequentes
- **Depoimentos** — feedbacks de clientes
- **Nossa história** — como a RefCoTech nasceu
- **Missão & Valores** — o que nos move
- **Contato** — canais de atendimento

### Materiais gratuitos (PDFs)
- `kit-referencias-refco.pdf` — paletas de cor, tipografias e inspirações de layout
- `checklist-portfolio-refco.pdf` — 32 itens para um portfólio profissional

---

## Tecnologias

- HTML5 + CSS3 + JavaScript (vanilla)
- Google Fonts: [Syne](https://fonts.google.com/specimen/Syne) + [Plus Jakarta Sans](https://fonts.google.com/specimen/Plus+Jakarta+Sans)
- [Formspree](https://formspree.io) — captação de e-mails
- PDFs gerados com [ReportLab](https://www.reportlab.com)
